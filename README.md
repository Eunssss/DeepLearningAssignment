HW1
1. 프로젝트 제목
뉴스 요약기 (News Summarizer)

2. 프로젝트 개요
이 프로젝트는 사용자가 뉴스 URL을 입력하면 해당 기사를 불러와 AI가 핵심 내용을 간단하게 요약해 주는 서비스이다.
긴 뉴스를 일일이 읽지 않고도 주요 정보를 빠르게 파악할 수 있도록 하였다.
Cursor(Gemini CLI)를 활용해 간단한 CLI 프로그램으로 구현했다.
